Job Portal Web Application
A full-stack job portal web application built using Java, JSP, Servlets, and JDBC. This platform enables job seekers to register, search for jobs, and apply, while employers can post jobs and manage applications. The project demonstrates core Java web development skills and showcases how job portals work in real-world scenarios.

I have developed this project by implementing critical job portal functionalities using my skills, and I am now ready to take on real-world projects confidently.

Features
User Registration & Login: Secure authentication for job seekers and employers.

Session Management: Tracks logged-in users and restricts unauthorized access.

Profile Management: Users can view and edit their profiles.

Job Listing: Employers can post jobs with details like title, description, location, and salary.

Job Search: Job seekers can search and filter jobs by keywords.

Job Application: Users can apply for jobs, and applications are tracked in the database.

Responsive UI: Clean and user-friendly interface using HTML, CSS, and Bootstrap.

Technologies Used
Frontend: HTML, CSS, JavaScript, JSP

Backend: Java Servlets, JDBC

Database: MySQL

Server: Apache Tomcat

IDE: Eclipse (or any Java IDE)

How to Run
Start the Tomcat server from Eclipse.

Open your browser and navigate to http://localhost:8080/JobPortal.

Register as a new user or log in with existing credentials.

Explore features like job search, application, and profile management.
